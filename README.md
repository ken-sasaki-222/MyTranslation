## アプリの概要
無駄な機能が一切無いシンプルな翻訳アプリ
[AppleStoreからインストール](#)

## 開発の経緯
App Storeからダウンロードできる翻訳アプリを多数使った結果、開発者が使いやすいと思えるアプリには出会えなかった。      
webアプリケーションではDeepLが最も使いやすく現在も愛用している。  
そこで、『使いづらいのなら自作しよう！』と考えたのが開発経緯です。  

使いやすいの定義は以下の3点  
・無駄がない  
・UIがシンプル  
・翻訳言語が限定されている

## 機能一覧
・翻訳機能（英語、日本語のみ）  
・翻訳内容読み上げ機能  
・翻訳履歴表示機能  
・お問い合わせ機能  
・レビュー機能  
・Twitter紹介機能（開発者）
